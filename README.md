# Sheikh Rayhan — Portfolio Website

Professional cybersecurity portfolio website for **Sheikh Rayhan** — Cyber Security Expert & Python Developer.

## Tech Stack
- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS 4**
- **shadcn/ui**
- **Framer Motion**

## Theme
Dark + Neon Green cybersecurity aesthetic with matrix rain animation, grid overlay, scanline effects, and neon glow borders.

## Sections
1. Navbar (sticky, responsive hamburger menu)
2. Hero (neon text, animated stats)
3. About Me (terminal card simulation)
4. Skills (6 cards with progress bars)
5. Projects (4 cards)
6. Security Tools (12 tools grid)
7. Courses (3 course cards)
8. Services & Hacking Methods (6 service cards)
9. Other Products (4 cards)
10. Contact (Telegram only — @SheikhRayhan404)
11. Footer

## Setup
```bash
npm install
npm run dev
```
